Manual Testing Instructions

  1. Start the development server: npm run dev
  2. Navigate to the registration page
  3. Fill out the registration form with valid data
  4. Submit the form
  5. Verify that the success message shows: "Registration successful! You can now proceed to login."
  6. Verify that the button shows: "Go to Login"
  7. Click the button and verify it redirects to the login page (/login)

After successful registration:

  • ✅ Button text says "Go to Login" instead of "Go to Homepage"
  • ✅ Button redirects to /login instead of /
  • ✅ Button CSS class is go-to-login-btn for proper styling
